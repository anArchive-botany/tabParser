TabParser

Sebbene questo programma possa somigliare molto a TabImport si comporta in maniera molto diversa.

Come linea di condotta generale non modifica i dati in automatico ma offre strumenti per poterlo fare agevolmente:

  • Per caricare un file bisogna trascinarlo sulla tabella
  • I pulsanti nella barra superiore hanno lo sfondo rosso se modificano i dati
  • Per modificare un dato basta fare click sulla casella
  • Per immettere particolari valori (es: una specie rilevata) è possibile fare click com il tasto destro su una casella e poi seleziona l'editor opportuno
  • Quando vengono modificati i dati dal programma le caselle cambiano colore, passandoci sopra con il mouse viene visualizzato il vecchio valore

Da tenere presente che:

  • Le celle con testo in grigio contengono dati che non verranno inviati al server
  • Le coordinate non vengono tradotte in automatico
  • Dopo l'importazione normalmente serve inserire i dati del progetto e i diritti
  • Facendo click su "salva" il file viene salvato nella stessa cartella dell'originale ma con un nuovo nome (aggiunge "-tabParser" e in formato ods), è possibile, ma sconsigliato, editare il file.
